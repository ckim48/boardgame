Game Development

Game Dev.

Objective

The objective of the game development team was to implement the back-end features of the digitized board game and guarantee that all game mechanics function properly. .

The game development team believed that: as we reside in the 3D space, it is our tendency to be much more deeply involved into a 3D virtual environment, and hence we expect the 3D features of the digitized board game to improve both visual and aural engagement within the players, leading to a more effective learning experience.

Also: By digitizing the boardgame, we are making the game available to every individual, regardless of their ethnicity, location, or age.

Game Design Team

Objective

The game design team had to first learn how to work from provided concept art, and to organize ideas before execution and production. .

Upon learning how to use Blender, created basic meshes to generate flat surface assets: Buildings and props and organic assets such as trees and rocks. And how to utilize reference inside of blender to assist in 3D modeling.
